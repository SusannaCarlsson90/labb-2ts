https://labb-2ts.onrender.com/

Jag har skapat en att göra applikation inför vår flytt.

Har använt vite för att snabbt starta upp, bygga och automatiskt uppdatera mitt projekt.

Använt git för versionshantering och publicerat sidan på Render.

Har använt TypeScript där jag bland annat skapat ett interface för hur en uppgift ska se ut och en klass som sköter all logik (lägga till, radera, spara).

Appen sparar listan i webbläsarens minne med hjälp av LocalStorage.

Man kan skriva in en uppgift, välja hur viktig den är (prio 1-3), bocka av den, radera den eller ångra om man råkar markera den som klar felaktigt.

Jag har stylat sidan med CSS för att den ska vara enkel att använda och se bra ut även på en mobiltelefon.
